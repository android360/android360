API Reference

  • Accelerometer\加速表

    点击进入该设备的运动传感器。

  • Camera

    Capture使用设备的相机拍照。

  • Capture

    Capture使用设备的媒体捕获应用程序的媒体文件。

  • Compass

    取得该设备指向的方向。

  • Connection

    快速检查网络状态，以及蜂窝网络信息。

  • Contacts

    与设备接触数据库的工作。

  • Device

    收集特定的设备信息。

  • Events

    挂接到通过JavaScript原生的事件。

  • File

    挂接到本地文件系统通过JavaScript 。

  • Geolocation

    让你的应用程序的位置感知。

  • Globalization

    启用特定于语言环境的对象表示。

  • InAppBrowser

    另一个在应用程序的浏览器实例启动的URL 。

  • Media

    录制和播放音频文件。

  • Notification

    视觉，听觉，和触觉设备的通知。

  • Splashscreen

    显示和隐藏应用程序启动画面。

  • Storage

    挂接到设备本地存储选项。
