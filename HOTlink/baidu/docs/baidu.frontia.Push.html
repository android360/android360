Methods

<static> send(data, options)

消息推送

Parameters:
Name Type Description
data Object

需要发送的数据

Properties
Name Type Description
messages String

指定消息内容，单个消息为单独字符串。如果有二进制的消息内容，请先做 BASE64 的编码。
msg_keys String

消息标识。指定消息标识，必须和messages一一对应。相同消息标识的消息会自动覆盖
options Object
Properties
Name Type Argument Description
push_type Number

推送类型，取值范围为：1～3 <ol> 1:单个人，必须指定user_id 和 channel_id （指定用户的指定设备）或者user_id（指定用户的所有设备）</br> 2:一群人，必须指定 tag </br> 3:所有人，无需指定tag、user_id、channel_id </br> </ol>
user_id String <optional>

用户标识，在Android里，channel_id + userid指定某一个特定client。不超过256字节，如果存在此字段，则只推送给此用户。
tag String <optional>

标签名称，不超过128字节
channel_id Number <optional>

通道标识
device_type Number <optional>

设备类型，取值范围为：1～5
message_type Number <optional>

消息类型 <ol> 0：消息（透传给应用的消息体） </br> 1：通知（对应设备上的消息通知）</br> 默认值为0。通知消息格式见example</br> </ol>
message_expires Number <optional>

指定消息的过期时间，默认为86400秒。必须和messages一一对应
success function <optional>

删除成功后callback
error function <optional>

删除失败后callback
