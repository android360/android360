RSS Feeds Reader Plugin for jQuery

This plugin will read RSS feeds from any website URL using the Google Feeds API. It produces structured HTML with in-built CSS classes for styling. Simple and easy to use.

特点

  • 支持 Atom 1.0, RSS 2.0, RSS 1.0 & RSS 0.9x.
  • For jQuery versions 1.4, 1.5, 1.6 and 1.7.
  • 指定饲料的数量，以回报.
  • 选择包括标题，日期和说明.
  • 指定标题 HTML tag.
  • 选择使用一个简短或完整描述（可用时）.
  • 生产的所有 HTML 代码和样式类。
  • 定制错误消息。
  • 支持SSL请求。
  • 指定饲料链接的目标。
  • 支持使用机箱标签媒体附件。
  • 用户回调函数。
  • 排序或日期。
  • 开始在指定的饲料项目。
  • 饲料重定向链接。
  • 格式饲料的日期。 * New *

入门

First include the jQuery and zRSSFeed libraries.

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js" type="text/javascript"></script>
<script src="jquery.zrssfeed.min.js" type="text/javascript"></script>

添加您希望的饲料来显示，并给它一个ID的DIV标记。没有必要指定一个类，这将增加。

<div id="test"></div>

现在添加脚本来调用zRSSFeed插件与 RSS URL 和任何选项。我们的例子中得到5饲料来自路透社的网站。

<script type="text/javascript">
$(document).ready(function () {
  $('#test').rssfeed('http://feeds.reuters.com/reuters/oddlyEnoughNews', {
    limit: 5
  });
});
</script>

Parameters

参数请求描述
url Yes 完整的 URL to the RSS feed ie http://feeds.reuters.com/reuters/oddlyEnoughNews
options No （见下文）的插件可选设置.
fn No 这是饲料后调用的可选用户回调函数被成功加载。

Plug-in Options/插件选项

参数默认描述
limit/限定 10 饲料的数量，返回（最多100个）.
offset/偏移 1 饲料项目编号开始在.
header true 如果为true ，包括含有饲料名称和链接的标题部分。
titletag/title标签 h4 指定HTML 标记为饲料称号。
date/日期 true 如果为true ，包括饲料的日期部分.
dateformat/日期格式 datetime 饲料日期的格式: 'datetime' - 完整的日期和时间, 'date' - 仅某天, 'time' - 仅时间, 'timeline' - lasped时间或格式掩码/mask ('dd/MM/yyyy hh:mm:ss').
content/内容 true 如果为true ，包括饲料的描述.
snippet/片段 true 如果为true ，显示了一个简短的描述为饲料项目，否则显示的全部内容。
media true 如果为true ，则显示媒体项目时可用。
showerror true 如果从谷歌订阅API是​​返回true ，和一个错误，则显示错误消息。
errormsg - 当发生错误时替换默认友好的消息。
key null 选择使用谷歌API密钥。
ssl false 支持SSL。使用安全的网页时设置为True 。
linktarget/连接对象 _self 指定所有饲料链接的目标 ('_blank', '_self', '_top', 框架name).
linkredirect/link重定向 - 指定页面重定向饲料链接。 原始链接被包含在 URL 命名参数 'link'.
linkcontent/link内容 false If true,显示饲料项目为连接的内容。
sort/分类 - 任何一方排序饲料项目 'title' or 'date'.留空为不排序。
sortasc true 指定排序方向， 升序(true) or 降序(false).
historical/历史 false If true, 返回任何可能存在的额外的历史记录。

HTML and CSS Styling

该插件将返回以下 HTML and styling classes. Note: The 'rssFeed' 类会自动添加到用户的元素。

成功返回

<div id="test" class="rssFeed">
  <div class="rssHeader"><
    <a>... (heading) ...</a>
  </div>
  <div class="rssBody"></div>
    <ul>
      <li class="rssRow odd">
        <h4><a>... (title) ...</a></h4>
        <div>... (date) ...</div>
        <p>... (description) ...</p>
        <div class="rssMedia">
          <div>Media files</div>
          <ul>
            <li><a>... (media link) ...</a></li>
          </ul>
        </div>
      </li>
      <li class="rssRow even">...</li>
      ...
    </ul>
  </div>
</div>

The 'odd' and 'even' 类可用于样式交替行。

失败的返回

<div id="test" class="rssFeed">
  <div class="rssError">
    <p>... (error message) ...</p>
  </div>
</div>
